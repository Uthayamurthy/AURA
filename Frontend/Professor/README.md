# AURA Professor Portal

The instructor's interface for managing classes and attendance in real-time.

## Features
-   **Class Management**: View assigned courses and schedules.
-   **Session Control**: Start and stop attendance sessions for a class.
-   **Live Monitoring**: See students checking in real-time.
-   **Manual Override**: Manually mark attendance if needed.

## Tech Stack
-   **Framework**: React (Vite)
-   **Language**: TypeScript
-   **Styling**: Tailwind CSS
-   **UI Components**: Shadcn UI (Radix Primitives)
-   **State Management**: React Query (TanStack Query) / Context API
-   **Routing**: React Router Dom
-   **Icons**: Lucide React

## Setup & Running

### Prerequisites
-   Node.js (v18+ recommended)
-   npm

### Installation
```bash
npm install
```

### Development Server
```bash
npm run dev
```
The app will typically run on `http://localhost:5174` (or similar port, distinct from Admin).

### Building
```bash
npm run build
```
